## 🐍 Introdução ao Python
Python é uma linguagem de programação de alto nível, interpretada, multiplataforma e de propósito geral, focada em simplicidade, legibilidade e produtividade. :contentReference[oaicite:0]{index=0}  

É muito utilizada tanto por iniciantes quanto por profissionais, pois permite escrever códigos claros e eficientes.

### Principais Características
- Sintaxe simples e fácil de entender  
- Alta produtividade  
- Grande comunidade e muitas bibliotecas  
- Forte uso no mercado e na educação  

### Histórico
- Criada por Guido van Rossum no final dos anos 80  
- Primeira versão em 1991  
- Python 3 (2008) é o padrão atual  
- Nome inspirado no grupo Monty Python  

### Áreas de Aplicação
- Desenvolvimento web  
- Automação  
- Ciência de dados  
- Machine Learning  
- Inteligência Artificial  
- Computação científica  

### Execução
- Modo interativo (linha a linha)  
- Scripts (.py)  

Exemplo:
print("Olá, mundo!")

## ⚙️ Conceitos Iniciais de Programação

O objetivo é entender a base da linguagem e criar programas simples. :contentReference[oaicite:1]{index=1}  

### Estrutura do Programa
- Execução de cima para baixo  
- Não precisa de função main  

Exemplo:
print("Meu primeiro programa")

---

### Comentários
- Uma linha: # comentário  
- Múltiplas linhas: """ comentário """  

---

### Variáveis
- Armazenam dados  
- Não precisam de tipo declarado  
- Tipagem dinâmica  

Exemplo:
nome = "Ana"  
idade = 20  
altura = 1.65  

---

### Tipos de Dados
- int → inteiro  
- float → decimal  
- str → texto  
- bool → verdadeiro/falso  

---

### Função type()
Verifica o tipo de um valor  

Exemplo:
print(type(idade))

---

### Entrada de Dados (input)
Recebe dados do usuário  

Exemplo:
nome = input("Digite seu nome: ")  

Obs: sempre retorna string  

---

### Conversão de Tipos
Exemplo:
idade = int(input())  
altura = float(input())  

---

### Operadores Aritméticos
+ soma  
- subtração  
* multiplicação  
/ divisão  
// divisão inteira  
% resto  
** potência  

Exemplo:
a = 10  
b = 3  
print(a + b)  
print(a * b)  
print(a ** b)  

---

### Boas Práticas
- Usar nomes claros (media_final)  
- Evitar acentos e espaços  
- Usar snake_case  

---

### Exercícios
1. Pedir nome e idade e exibir mensagem  
2. Somar dois números  
3. Calcular média de três notas
