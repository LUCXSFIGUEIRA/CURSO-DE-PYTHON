# 📘 Resumo – Introdução ao Python + Conceitos Iniciais

## 🐍 Introdução ao Python

Python é uma linguagem de programação **de alto nível, interpretada, multiplataforma e de propósito geral**, conhecida pela sua **simplicidade e legibilidade**, sendo ideal tanto para iniciantes quanto para aplicações profissionais. :contentReference[oaicite:0]{index=0}  

### 🔹 Principais Características
- Sintaxe simples e próxima da linguagem humana
- Alta produtividade (menos código para resolver problemas)
- Grande comunidade e vasto ecossistema
- Muito utilizada na indústria e no meio acadêmico

### 🔹 Áreas de Aplicação
- Desenvolvimento web  
- Automação de tarefas  
- Ciência de dados  
- Inteligência artificial e Machine Learning  
- Computação científica  

### 🔹 Histórico
- Criada por **Guido van Rossum** no final dos anos 80  
- Primeira versão em 1991  
- Python 3.0 (2008) é o padrão atual  

### 🔹 Execução
- Modo interativo (linha a linha)  
- Scripts `.py`
